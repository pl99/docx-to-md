| Код    | Функция                                                      | Описание                                                                                                                                                                                                                                                                    |
| ------ | ------------------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Ф-01.1 | Создание заявки на онлайн-видеопоток                         | Создание заявки типа «онлайн просмотр» с указанием информации о пользователе, создавшем заявку, данных о ТС, камеры и места размещения камеры. Одна заявка соответствует одной камере.                                                                                      |
| Ф-01.2 | Создание заявки на архивный видеопоток                       | Создание заявки типа «запрос архива» с указанием информации о пользователе, создавшем заявку, данных о ТС, камеры и места размещения камеры, а также периода, за который требуется видеоархив. Одна заявка соответствует одной камере.                                      |
| Ф-01.3 | Приём заявки по API от ЕЦХД                                  | Создание заявки по обращению ЕЦХД через API.                                                                                                                                                                                                                                |
| Ф-01.4 | Валидация атрибутов при создании заявки                      | Проверка обязательных полей, форматов дат, данных о ТС и камерах. В случае ошибок пользователю выводится информационное сообщение с описанием ошибок. При поступлении заявки от внешней системы заявка не создаётся, а информация об ошибках записывается в лог Подсистемы. |
| Ф-01.5 | Присвоение идентификатора и метки времени, сохранение заявки | Генерация уникального идентификатора заявки, фиксация даты и времени создания, сохранение заявки в системе.                                                                                                                                                                 |
